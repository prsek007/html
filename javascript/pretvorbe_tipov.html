Pretvorbe tipa spremenljivke

Poznamo različne tipe spremenljivk npr: celo število (Int), decimalno število (Float), besedilo (String), prav narobe (Boolean)... Pogosto moramo pretvoriti iz enega tipa v drug tip.

Pretvorba v niz (String)

String() funkcija:

let stevilo = 123; let niz = String(stevilo); // "123"

toString() metoda:

let stevilo = 123; let niz = stevilo.toString(); // "123"

Pretvorba v število (Number)

Number() funkcija:

let niz = "123"; let stevilo = Number(niz); // 123

parseInt() funkcija:

let niz = "123"; let stevilo = parseInt(niz); // 123

parseFloat() funkcija:

let niz = "123.45"; let stevilo = parseFloat(niz); // 123.45

Pretvorba v boolean (Boolean)

Boolean() funkcija:

let niz = "true"; let bool = Boolean(niz); // true

Samodejna pretvorba (Type Coercion)

JavaScript lahko samodejno pretvori tipe spremenljivk v določenih situacijah.

Primer:

let stevilo = 123; let niz = "456"; let rezultat = stevilo + niz; // "123456" (številka je pretvorjena v niz)

Primeri uporabe

// Pretvorba niza v število let niz = "123"; let stevilo = Number(niz); // 123 // Pretvorba števila v niz let stevilo2 = 456; let niz2 = String(stevilo2); // "456" // Pretvorba niza v boolean let niz3 = "true"; let bool = Boolean(niz3); // true
