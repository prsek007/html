Spremenljivke

Spremenljivke so osnovni gradniki programiranja, ki jih uporabljamo za shranjevanje podatkov. Predstavljaj si jih kot škatle, v katere lahko spraviš različne vrednosti, kot so številke, besedila ali logične vrednosti. Tukaj je nekaj ključnih točk o spremenljivkah:

Definiranje spremenljivk

V JavaScriptu lahko spremenljivke definiramo z var, let ali const.

let ime = "Jaka"; // Uporabimo let za spremenljive vrednosti const starost = 13; // Uporabimo const za nespremenljive vrednosti

Uporaba spremenljivk

Ko enkrat definiramo spremenljivko, jo lahko uporabimo v naši kodi.

console.log("Ime: " + ime); // Ime: Jaka console.log("Starost: " + starost); // Starost: 13

Spreminjanje vrednosti

Spremenljivkam, definiranih z let ali var, lahko spremenimo vrednost.

ime = "Marko"; console.log("Novo ime: " + ime); // Novo ime: Marko

Pravila za poimenovanje

Imena spremenljivk morajo začeti s črko, podčrtajem (_) ali dolarjem ($).

Ne smejo vsebovati presledkov ali posebnih znakov (razen podčrtaja in dolarja).

let _mojaSpremenljivka = 10; let $cena = 99.99;
