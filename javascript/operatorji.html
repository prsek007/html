Operatorji

V JavaScriptu obstaja več vrst operatorjev, ki se uporabljajo za različne matematične in logične operacije. Tukaj je pregled glavnih vrst operatorjev:

Aritmetični operatorji

Addition (+): Seštevanje dveh števil.

let a = 5 + 3; // a bo 8

Concatenation: Združevanje nizov ali niza s številom

let a = "5" + 3; // a bo '53' !! niz, ne število let ime = "Miha" + " " + "13" + "let"; // ime bo Miha 13 let

Subtraction (-): Odštevanje enega števila od drugega.

let b = 5 - 3; // b bo 2

Multiplication (*): Množenje dveh števil.

let c = 5 * 3; // c bo 15

Division (/): Deljenje enega števila z drugim.

let d = 15 / 3; // d bo 5

Modulus (%): Ostanek pri deljenju.

let e = 5 % 2; // e bo 1

Exponentiation ()**: Potenciranje.

let f = 2 ** 3; // f bo 8

Prireditveni operatorji

Assignment (=): Prireditev vrednosti spremenljivki.

let x = 10;

Addition Assignment (+=): Dodajanje in prireditev.

x += 5; // x bo 15

Subtraction Assignment (-=): Odštevanje in prireditev.

x -= 3; // x bo 12

Multiplication Assignment (*=): Množenje in prireditev.

x *= 2; // x bo 24

Division Assignment (/=): Deljenje in prireditev.

x /= 4; // x bo 6

Primerjalni operatorji

Equal to (==): Preverja, ali sta dve vrednosti enaki.

5 == '5'; // true

Strict Equal to (===): Preverja, ali sta dve vrednosti enaki in istega tipa.

5 === '5'; // false

Not Equal to (!=): Preverja, ali dve vrednosti nista enaki.

5 != '5'; // false

Strict Not Equal to (!==): Preverja, ali dve vrednosti nista enaki ali nista istega tipa.

5 !== '5'; // true

Greater than (>): Preverja, ali je ena vrednost večja od druge.

5 > 3; // true

Less than (<): Preverja, ali je ena vrednost manjša od druge.

5 < 3; // false 4.

Logični operatorji

AND (&&): Vrača true, če sta oba izraza true.

true && false; // false

OR (||): Vrača true, če je vsaj eden od izrazov true.

true || false; // true

NOT (!): Vrača nasprotno vrednost izraza.

!true; // false

Ternarni operator

Condition ? expr1 : expr2: Vrača expr1, če je pogoj true, sicer expr2.

let rezultat = (5 > 3) ? 'večje' : 'manjše'; // rezultat bo 'večje'
