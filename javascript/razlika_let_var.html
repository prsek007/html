Razlika med var in let

Razlika med var in let v JavaScriptu je pomembna, saj vpliva na obseg (scope) in vedenje spremenljivk.

Nekaj ključnih razlik

Obseg (Scope)

var ima funkcijski obseg (function scope), kar pomeni, da je spremenljivka dostopna znotraj funkcije, v kateri je definirana.

function testVar() { if (true) { var x = 10; } console.log(x); // 10 (x je dostopen zunaj if bloka) }

let ima blokovni obseg (block scope), kar pomeni, da je spremenljivka dostopna samo znotraj bloka kode, kjer je definirana (npr. znotraj {}).

function testLet() { if (true) { let y = 10; } console.log(y); // ReferenceError: y is not defined (y ni dostopen zunaj if bloka) }

Ponovno definiranje

Spremenljivke, definirane z var, je mogoče ponovno definirati znotraj istega obsega.

var a = 1; var a = 2; // Veljavno

Spremenljivke, definirane z let, ni mogoče ponovno definirati znotraj istega obsega.

let b = 1; let b = 2; // SyntaxError: Identifier 'b' has already been declared

Dviganje (Hoisting):

Spremenljivke, definirane z var, so dvignjene (hoisted) na vrh njihovega obsega in inicializirane z undefined.

console.log(c); // undefined var c = 3;

Spremenljivke, definirane z let, so prav tako dvignjene, vendar niso inicializirane. Dostop do njih pred inicializacijo povzroči napako.

console.log(d); // ReferenceError: Cannot access 'd' before initialization let d = 3;

Te razlike so pomembne za razumevanje, kako se spremenljivke obnašajo v različnih delih kode in kako se izogniti napakam. Če imaš še kakšno vprašanje ali potrebuješ dodatne primere, mi sporoči!
