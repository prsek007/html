Console log

console.log() je metoda v JavaScriptu, ki se uporablja za izpisovanje sporočil v konzolo brskalnika. To je zelo uporabno orodje za razhroščevanje (debugging) kode, saj omogoča programerjem, da preverijo vrednosti spremenljivk, rezultate funkcij in druge pomembne informacije med izvajanjem programa.

Kako do konzole

Konzolo lahko odpremo s tipko F12 na tipkovnici ali z desnim klikom na spletni strani in nato Preglej Zgoraj izberemo jeziček Console.

Primeri uporabe

Izpisovanje besedila:

console.log("Pozdravljen, svet!"); // Izpiše: Pozdravljen, svet!

Izpisovanje vrednosti spremenljivk:

let ime = "Jaka"; console.log(ime); // Izpiše: Jaka

Izpisovanje rezultatov izrazov:

let a = 5; let b = 10; console.log(a + b); // Izpiše: 15

Izpisovanje objektov in polj:

let oseba = { ime: "Jaka", starost: 13 }; console.log(oseba); // Izpiše: { ime: "Jaka", starost: 13 } let stevila = [1, 2, 3, 4, 5]; console.log(stevila); // Izpiše: [1, 2, 3, 4, 5]

console.log() je zelo uporabna metoda za sledenje toku programa in iskanje napak. Če imaš še kakšno vprašanje ali potrebuješ dodatne primere, mi sporoči!
