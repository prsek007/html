Vejitev

Vejitev (pogojni stavki) v programiranju omogoča, da program sprejema odločitve in izvaja različne dele kode glede na določene pogoje. V JavaScriptu se za vejitev najpogosteje uporabljajo stavki if , else if in else.

Primeri

if stavek

Uporablja se za preverjanje pogoja. Če je pogoj resničen (true), se izvede koda znotraj bloka if.

let starost = 18; if (starost >= 18) { console.log("Polnoleten si."); } // Izpiše: Polnoleten si.

else stavek

Uporablja se skupaj z if stavkom za definiranje kode, ki se izvede, če pogoj v if stavku ni resničen (false).

let starost = 16; if (starost >= 18) { console.log("Polnoleten si."); } else { console.log("Nisi še polnoleten."); } // Izpiše: Nisi še polnoleten.

else if stavek

Uporablja se za preverjanje dodatnih pogojev, če prejšnji pogoji niso resnični.

let ura = 15; if (ura < 12) { console.log("Dobro jutro!"); } else if (ura < 18) { console.log("Dober dan!"); } else { console.log("Dober večer!"); } // Izpiše: Dober dan!

Gnezdeni pogoji

Pogoje lahko tudi gnezdimo, kar pomeni, da imamo en pogoj znotraj drugega.

let ura=10; let dan="ponedeljek"; if (ura < 12) { if (dan === "ponedeljek" ) { console.log("Dobro jutro, ponedeljek!"); } else { console.log("Dobro jutro!"); } } else { console.log("Dober dan!"); } // Izpiše: Dobro jutro, ponedeljek!

ternarni operator

Ternarni operator, znan tudi kot pogojni operator, je kratek način za zapisovanje preprostih pogojnih stavkov v JavaScriptu. Uporablja se lahko namesto if-else stavkov, kadar želimo zapisati pogoj in dve možni vrednosti v eni vrstici. Sintaksa ternarnega operatorja je:

pogoj ? vrednost_če_resnično : vrednost_če_napačno;

Primeri

Preprosti pogoji

Ko imamo preprost pogoj, ki določa eno od dveh vrednosti.

let starost = 18; let status = (starost >= 18) ? "Polnoleten" : "Mladoleten"; console.log(status); // Izpiše: Polnoleten

Privzete vrednosti

Ko želimo določiti privzeto vrednost, če je spremenljivka undefined ali null.

let ime = null; let prikazanoIme = ime ? ime : "Gost"; console.log(prikazanoIme); // Izpiše: Gost

Kratki izrazi

Ko želimo ohraniti kodo kratko in berljivo.

let ura = 10; let pozdrav = (ura < 12) ? "Dobro jutro" : "Dober dan"; console.log(pozdrav); // Izpiše: Dobro jutro

Vrednosti v funkcijah

Ko želimo vrniti vrednost na podlagi pogoja znotraj funkcije.

function preveriStarost(starost) { return (starost >= 18) ? "Polnoleten" : "Mladoleten"; } console.log(preveriStarost(20)); // Izpiše: Polnoleten

Ternarni operator je zelo uporaben za preproste pogoje, saj omogoča krajši in bolj jedrnat zapis. Vendar pa ga je bolje uporabiti le za enostavne primere, saj lahko preveč zapleteni ternarni izrazi zmanjšajo berljivost kode. Če imaš še kakšno vprašanje ali potrebuješ dodatne primere, mi sporoči!
