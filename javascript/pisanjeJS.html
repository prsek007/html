Osnovna navodila

Podobno kot css, lahko JavaScript kodo pišemo na različnih mestih, znotraj značke script v glavi dokumenta, v vrstici ali v telesu dokumenta (body) na koncu strani ali kot zunanjo datoteko.

V glavi dokumenta

V tem primeru damo značko script v glavo dokumenta in znotraj tega napišemo kodo.

<html> <head> <title>Naslov strani</title> <script> // JavaScript koda </script> </head> <body> </body> </html>

V vrstici

V tem primeru kar v vrstici, kjer želimo, da se koda izvrši, kodo napišemo, prav tako znotraj značke script

<html> <head> <title>Naslov strani</title> </head> <body> <p>Tukaj je JavaScript koda <script> // JavaScript koda </script></p> </body> </html>

V telesu dokumenta, na koncu

V tem primeru, ki se koda znotraj značke script piše pod celotno kodo html, tik pred zaprtjem značke </body>

<html> <head> <title>Naslov strani</title> </head> <body> <p>Celotna vsebina html dokumenta.</p> <script> // JavaScript koda </script> </body> </html>

Zunanja datoteka

Najpogosteje se uporablja pisanje JavaCsript kode v zunanji datoteki, saj to med drugim omogoča večjo preglednost html strani ter uporabo iste JavaScript kode na večih dokumentih.

<html> <head> <title>Naslov strani</title> <script src="script.js"></script> </head> <body> </body> </html>

Osnovna sintaksa

Sintaksa so pravila pisanja kode, ki jih zahteva določeni programski jezik. Pri javaScriptu, tako kot pri html in css lahko vse ukaze pišemo v eno vrstico vendar je vsak programski stavek od naslednjega ločen s podpičjem. Tako pisanje je sicer dovoljeno vendar nikakor ni priporočljivo saj je popolnoma nepregledno in je zelo težko iskati morebitne napake.

Podpičje
Vsak programski stavek zaključimo s podpičjem var i = 10;.
Priporoča se, da je vsak stavek v svoji vrstici.
Blok
Vsak blok kode je znotraj zavitih oklepajev { // blok kode }.
Priporoča se, da je koda v bloku zamaknjena za eno tabulatorsko mesto oz. 3-5 presledkov, odvisno od nastavitev programa za pisanje kode.
Imena spremenljivk
Imena spremenljivk se morajo začeti s črko, podčrtajem ali znakom dolar ($) in ne smejo vsebovati posebnih znakov (@[]>!"#$% itd.) razen podčrtaja, prav tako ime spremenljivke ne sme vsebovati čšž in presledkov.
Priporoča se, da je ime smiselno in spominja na vrednosti, ki so v njej hranjene ter da se začne z malo črko, vsaka naslednja beseda pa z veliko črko (camelCase) npr: var danesJeLepSoncenDan = true;
Funkcije
Vsaka funkcija ima ime in oklepaje danVTednu().
Za imena velja enako kot za spremenljivke.

Primer

<script> function stejDoDeset() { for (let i = 1; i < 11; i++) { console.log(i); } } </script>

Osnovna sintaksa

Spremenljivke

let ime = "Janez"; const starost = 25;

Funkcije

function pozdrav() { alert("Pozdravljeni!"); }

Dogodki

JavaScript lahko uporabite za obdelavo dogodkov, kot so kliki, premiki miške itd.

Primer

<button onclick="pozdrav()">Klikni me</button>

Manipulacija DOM (Document Object Model)

JavaScript omogoča spreminjanje vsebine HTML dokumenta.

Primer

document.getElementById("mojElement").innerHTML = "Nova vsebina";
