SCSS (sass)

Razširjen css oz. scss omogoča tudi dodajanje spremenljivk. Za razliko od css datoteke, moramo dedaj datoteka s slogom shraniti s končnoco scss npr. slog.scss. Spremenljivke dodamo vedno preden jo uporabimo, občajno na vrhu s prespono $ (dolar) npr. $barva: red;, potem pa lahko to spremenljivko uporabimo namesto vrednosti npr. 

    $barva: red;
    ol {
        color: $barva;
    }
    ul {
        color: $barva;
    }

Compile

Ka datoteko shranimo, jo povežemo z dokumentom tako kot smo vajeni z .css datoteko <ling rel="stylesheet" href="slog.css"> vendar to še ne bo delovalo ker:

  1. datoteke z imenom slog.css sploh nimamo ampak imamo datoteko slog.scss
  2. tidi če bi vnesli <ling rel="stylesheet" href="slog.scss"> ne bi delovalo, ker je brskalniki ne podpirajo
  3. ker moramo datoteko slog.scss pretvoriti (prekompajlati) v datoteko slog.css

Ustvari task.json

V delovni mapi ustvarimo mapo .vscode, VS Code pa ustvarimo tasks.json datoteko navodila.

Vsebina datoteke tasks.json za scss datoteko

Zaženi Terminal > Configure Tasks and click Create tasks.json file from template ter vstavi sledečo kodo v dokument.

S Ctrl*Shift+B zaženemo tascs.json

// Sass configuration
{
// See https://go.microsoft.com/fwlink/?LinkId=733558
// for the documentation about the tasks.json format
"version": "2.0.0",
"tasks": [
        {
            "label": "Sass Compile",
            "type": "shell",
            "command": "sass styles.scss styles.css",
            "group": "build"
        }
    ]
}

Dedovanje

%sporocilo {
    font-family: sans-serif;
    font-size: 18px;
    font-weight: bold;
    border: 1px solid black;
    padding: 20px;
    margin: 20px;
}   
.uspeh {
    @extend %sporocilo;
    background-color: lightgreen;
    border: green;
}   
.napaka {
    @extend %sporocilo;
    background-color: rgb(248, 140, 167);
    border: red;
}   
.opozorilo {
    @extend %sporocilo;
    background-color: lightcoral;
    border: orange;
}
