CSS

Običajni odstavek.

CSS (Cascading Style Sheet) je namenjen oblikovanju html dokumenta. Pomembno je, da lahko dodajamo obliko oz. slog na različnih nivojih (cascading). Končna oblika je sestavljena iz vseh predhodnih nastavitev, obvelja pa tista, ki je nastavljena na zadnje, na najvišjem nivoju.

Slog znotraj značke

To je najvišji nivo. Recimo, da bi naslednji odstavek radi imeli v modri pisavi. To nastavimo tako, da dodamo atribut style, znak =, narekovaje "" in znotraj narekovajev določomo barvo color: blue takole <p style="color: blue;">. Za color je dvopičje in nato vrednost (blue).

<p style="color: blue;">Modra pisava</p>

Določimo lahko več nastavitev na primer sredinsko poravnavo (text-alingn: center). Več nastavitev ločimo s podpičjem ;.

<p style="color: green; text-align: center;">Zelena pisava, poravnava na sredino</p>

Znotraj značke style

Če bi radi oblikovali spletno stran tako, da večkrat uporabljamo enako oblikovanje, ni pa smiselno, da enako kodo večkrat kopiramo, oblikovanje določimo znotraj značke <style>, ki je postavimo znotraj značke <head> na začetku html kode. Sintaksa je malce drugačna in sicer naprej napišemo ime značke, nastavitve pa podajamo znotraj zavitih oklepajev. <style> h2 {color: orange; text-align: center; }</style>.

<style> ul li {color: red; list-style-type: square; }</style>.

Zunanja datoteka

Zunanja datoteka ima kar nekaj prednosti:

  • ni sloga na začetku spletne strani, zato je sama stran krajša in bolj pregledna
  • isti slog lahko uporabim za več spletnih strani
  • z eno spremembo lahko spremenim izgled na vseh spletnih straneh
  • vse strani, ki so povezave z istim slogom, imajo enak izgled

stran z zunanjim slogom vdelam zelo preprosto. Naredim spletne dokument s končnico css npr. slog.css in v spletni strani, v glavi dodam <link src="stylesheet" href="slog.css">. V to datoteko pa shranim sloge, kot če bi jih pisal v znački <style>body {background-color: black;}</style> in tako naprej, le da značke style sploh ne pišemo.

Primer:

V datoteko slog.css shranimo tako vsebino.

body { background-color: antiquewhite; color: rgb(27, 27, 27) ; } h2 { color: orange; text-align: center; } ul li { color: red; list-style-type: square; } ul li { color: red; list-style-type: square; }

Oblika pisanja

Navodila za css, podobmno kot html lahko pišemo vsa v eno vrstico, eno pod drugo ali kakor želimo, pomembno je, da vsako značko najprej navedemo, potem pa v zavitih oklepajih navajamo lastnosti, vrednosti ločimo z dvopičjem, lastnosti pa s podpičjem.

Priporočilo

Ne glede na prejšnjo trditev, pa obstajapriporočilo, ki pravi, da je najbolje, da ime značke in nato vsako lastnost posebej damo v svojo vrstco, lastnosti še malo zamaknemo (za en o tabulatorsko mesto), da je vse skupaj dosti bolj pregledno (glej primer zgoraj).

Multiple selector - , (vejica)

Hkrati lahko vplivamo naveč elementov tako, da med različne značke dodamo vejico npr. th, td { border: colod 3px gray; } določi siv okvir debeline 1px za oba elementa th in td

id - # (lojtra)

Elementu lahko določimo id, kar je primerno, če bi le enemu določenemu elementu želeli dodali obliko ali povezavo nanj. Za sklic na ta element uporabimo znak lojtra in id elementa.

Primer

<style> #odstavek1 { color: rgb(250, 250, 250); font-size: 18px; border: solid 2px rgb(123, 81, 2); background-color: rgb(3, 117, 174); padding: 5px; text-align: center; } </style>

<p id="odstavek1">Odstavek bela pisava, velikost pisave 18px, rjav rob 2px, modro ozadje, odmik 5px, besedilo poravnano na sredino.</p>

class - . (pika)

Bolj pogosto kot id, uporabljamo class vendar class običajno uporabljamo, ko želimo enak slog dodati večim elementom hkrati.

Primer:

<div class='codeDiv'> <style> #odstavek1 { color: rgb(250, 250, 250); font-size: 18px; border: solid 2px rgb(123, 81, 2); background-color: rgb(3, 117, 174); padding: 5px; text-align: center; } </style>
