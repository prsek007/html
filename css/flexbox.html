Flexbox

Za prikaz večihlementov vzporedno, ki se prelomijo v naslednjo vrstico, če je ožji zaslon. Uporabimo div element, ki mu določimo class npr. container <div class="container" ter mu določimo css lastnosti:

    .container {
        display: flex;
        flex-wrap: wrap;
        }

S tem smo povedali, da naj se ta div element prikazuje kot flexbox element display: flex; ter da se naj elementi samodejno prelamljajo v novo vrstico flex-wrap: wrap;. Znotraj tega div elementa pa dodamo div elemente z vsebino. Tudi tem lahko določomo lastnosti npr. takole: 

    .container > div {
        background-color: rgb(5, 133, 172);
        font-size: 20px;
        margin: 20px;
        padding: 20px;
        color: white;
        width: 200px;
    }
