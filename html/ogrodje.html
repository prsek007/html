Ogrodje

Ogrodje se uporablja za začetek. Preprosto prekopiramo vsebino od <!DOCTYPE naprej in shranimo v novo html datoteko, priredimo in to je to.

<!DOCTYPE html>
<html lang="sl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ogrodje spletne strani</title>
    <link rel="stylesheet" href="style.css"> <!-- Povezava na zunanji CSS -->
    <style>
    /* slog znotraj datoteke */
        body {
            background-color: antiquewhite;
            color: rgb(27, 27, 27) ;
        }
    </style>

</head>
<body>
    <h1>Dobrodošli na moji spletni strani!</h1>
    <p>To je osnovna HTML stran.</p>

    <script src="script.js"> // Povezava na zunanji JS
        // tukaj pride JavaScript koda
    </script>
</body>
</html>
